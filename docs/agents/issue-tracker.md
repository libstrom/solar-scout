# Issue Tracker

Issues live in **GitHub Issues** for this repository (`libstrom/edek-parser`).

## CLI tool

Use the `gh` CLI for all issue operations.

## Common commands

```bash
gh issue create --title "..." --body "..." --label "needs-triage"
gh issue list
gh issue view <number>
gh issue edit <number> --add-label "ready-for-agent"
gh issue close <number> --comment "..."
```

## Notes

- Always prefix AI-generated triage comments with `> *This was generated by AI during triage.*`
