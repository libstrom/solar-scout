# Domain Docs

## Layout

**Single-context** — one `CONTEXT.md` at the repo root, ADRs in `docs/adr/`.

## CONTEXT.md

- Location: `/CONTEXT.md`
- Create lazily when the first term is resolved.

## ADRs

- Location: `/docs/adr/`
- Naming: `0001-decision-name.md`
- Create lazily for hard-to-reverse decisions.
