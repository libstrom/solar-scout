🎯 Best practice — vad marknadsledarna gör

  • Fullskärmskarta som primär canvas — karta tar 60–80% av skärmen, inte inbäddad i kolumn
  • Satellit-first — alla börjar på satellitvy, inte gatukarta. Rita område → scan direkt på bilden.
  • Leads som overlay på kartan — färgkodade pins, klicka pin → detaljer i sidopanel. Ingen tabswitch.
  • Värmekarta / heatmap — Aurora Solar visar irradiansöverlapp direkt på taket (gult = hög potential)
  • Panel-sidebar, inte separata flikar — CRM-vy (ringlista) alltid synlig bredvid kartan
  • Ett enda sökfält, smart auto-detect — stad → scan, adress → geocode, koordinater → pan, postnummer → zoom. Som Google Maps / Eniro / Hitta.se.
  • Snabb statusuppdatering — ett klick ändrar lead-status, auto-sparat, ingen modal
