Vad finns i filen
Kolumn för kolumn
🏠
fastighetsbeteckning
Officiell beteckning ur energideklarationen. Länken mot kundregistret.
ex: "Höllviken 20:98"
📍
adress / postnummer / ort
Fastighetens gatuadress extraherad ur PDF eller XLSM-fil.
ex: "Möllevägen 24A, 23642 Höllviken"
⚡
energiklass
A–G enligt Boverkets skala. D/E/F/G = stor förbättringspotential.
ex: "E" → sol+VP ger störst effekt
🌡️
uppvarmningssystem
Nuvarande värmekälla. El direkt och olja är dyrast = bäst pitch.
ex: "El (direktverkande)"
🎯
score
0–100. Baserat på energiklass, förbrukning, uppvärming och solpotential. Sortera på denna.
≥80 = ring idag
📦
bucket
SOL_BATTERI eller BATTERI. Direkt produktrekommendation utan analys.
BATTERI = de har sol redan
💬
pitch_points
Färdiga säljargument baserade på fastighetsdata. Läs innan samtalet.
ex: "Hög elförbrukning 19 830 kWh/år…"
👤
reg_namn / reg_telefon / reg_mejl
Kontaktinfo från kundregistret. Finns bara för gröna rader (match).
ex: "Sandra Larsson, 073-445 67 76"
📅
reg_senaste_kontakt
Senaste datum vi hade kontakt med kunden ur FileMaker.
ex: "2023-10-05"
🏷️
reg_status
Köpare = bor där nu (prioritet). Säljare = har lämnat fastigheten.
"Köpare" > "Säljare" alltid
🔗
match_tier / match_confidence
match / verifiera / ingen. Confidence 0–100% för fuzzy-träffar.
"verifiera" = kolla manuellt
☀️
har_solceller / har_solvarme
Har de redan sol? Då är BATTERI rätt produkt, inte SOL_BATTERI.
true → bucket = BATTERI
